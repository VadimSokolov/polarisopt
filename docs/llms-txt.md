# llms.txt — compact summary for LLM context budgets

This is the polarisopt project summary in the
[llms.txt](https://llmstxt.org/) format — flat plain text under ~8k
tokens, designed to fit in an LLM context window so AI tools can pick
up the project without having to crawl the full docs.

The file lives at [`docs/llms.txt`](llms.txt) for tools that follow
the convention of looking at the repo root.
