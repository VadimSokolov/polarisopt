# Tutorials

Step-by-step walkthroughs. Each tutorial is self-contained and copy-paste runnable.

| # | Tutorial | What you'll learn |
|---|---|---|
| 01 | [Branin demo](01-branin-demo.md) | Run a 5-minute Bayesian-optimization study on a benchmark function. No POLARIS install required. |
| 02 | [Morris screening](02-morris-screening.md) | Use a static design to rank parameter importance. |
| 03 | [Multi-objective](03-multi-objective.md) | Sequential BO with q-EHVI on two competing objectives. |
| 04 | [Restart](04-restart.md) | Kill a study mid-flight, resume it, confirm RNG continuity. |
| 05 | [First POLARIS run](05-first-polaris.md) | Wire `PolarisSimulator` to a real model on a Slurm cluster. |
| 06 | [Writing a plugin](06-write-a-plugin.md) | Add a custom design via the plugin registry. |

If you're brand-new to polarisopt, start with **01 · Branin demo** —
it's the fastest path from `pip install` to a working BO loop.
